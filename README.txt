# Classroom Lab App

## Author

Lee Kusowski

## Description

PRODUCTION ONLY

The Classroom Lab App is a django-based web app meant to support access to
virtual machines (VM's) to suppliment CIS course materials.

    Users can:
Signup & Login
Be members of a Classroom
View a Classroom's Exercises
Open permitted Exercises
Complete a Submission for a Task

    Instructors can:
Create Classes
Create Exercises
Create Tasks

    Admin can:
CRUD for Users
